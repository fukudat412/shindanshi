# 問題テンプレート集

このドキュメントは、1つの論点（topic）から複数の問題を量産するためのテンプレートを定義します。

---

## 1. 問題生成の基本ルール

### 1.1 1論点 → 5〜7問に分解する
各論点は最低でも以下の問題を揃える：

| # | タイプ | フェーズ | 説明 |
|---|--------|----------|------|
| 1 | 定義確認 | INTRO | 用語の定義を○×または短文で確認 |
| 2 | 判断基準 | UNDERSTAND | 4択で正しい判断基準を選ぶ |
| 3 | 手順の順序 | UNDERSTAND | 正しい手順や順序を選ぶ |
| 4 | ありがちな誤解 | RETAIN | ひっかけ問題（よくある間違いを問う） |
| 5 | 逆方向 | RETAIN | 原因→結果、または結果→原因を問う |
| 6 | ミニケース | EXAM | 短い文章を読んで判断する |
| 7 | 二次接続 | EXAM | 論点を言語化/助言化する（二次対策） |

### 1.2 フェーズの定義
- **INTRO（導入）**: 基礎的な定義確認。初学者向け
- **UNDERSTAND（理解）**: 判断基準や手順の理解を確認
- **RETAIN（定着）**: ひっかけ問題で定着度を確認
- **EXAM（試験対策）**: 実践的な問題で試験対応力を確認

## 2. 一次試験向けテンプレート

### 2.1 定義確認（○×/短文）
```
【フェーズ】INTRO
【タイプ】TRUE_FALSE または SHORT_TEXT

【テンプレート】
「{用語}とは、{定義}である。」→ ○×
「{用語}の定義を答えよ。」→ 短文

【例：NPV】
問題：「NPV（正味現在価値）とは、投資から得られる将来キャッシュフローの現在価値の合計から初期投資額を差し引いた値である。」
答え：true
解説：NPVは将来のキャッシュフローを現在価値に割り引き、初期投資を差し引くことで投資の価値を評価する指標です。
```

### 2.2 判断基準（4択）
```
【フェーズ】UNDERSTAND
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「{論点}において、正しい判断基準はどれか。」

【例：NPV】
問題：「投資判断においてNPVを用いる場合、正しい基準はどれか。」
選択肢：
A. NPVがプラスなら採用
B. NPVがマイナスなら採用
C. NPVが0なら必ず採用
D. NPVの大きさは関係ない
答え：A
解説：NPV > 0 の投資案は、投資額以上のリターンが期待できるため採用すべきです。
```

### 2.3 手順の順序（4択）
```
【フェーズ】UNDERSTAND
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「{論点}の計算/実施手順として正しいものはどれか。」

【例：NPV】
問題：「NPV計算の正しい手順はどれか。」
選択肢：
A. 将来CF予測 → 現在価値に割引 → 初期投資を差引
B. 初期投資を差引 → 将来CF予測 → 現在価値に割引
C. 現在価値に割引 → 初期投資を差引 → 将来CF予測
D. 将来CF予測 → 初期投資を差引 → 現在価値に割引
答え：A
解説：NPV計算は、1.将来CFを予測、2.割引率で現在価値に換算、3.初期投資を差し引く、の順で行います。
```

### 2.4 ありがちな誤解（ひっかけ4択）
```
【フェーズ】RETAIN
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「{論点}について、誤っているものはどれか。」
「{論点}のよくある誤解として正しいものはどれか。」

【例：NPV】
問題：「NPVについて、誤っているものはどれか。」
選択肢：
A. NPVは時間価値を考慮している
B. NPVがプラスの案は採用すべき
C. NPVが最大の案を常に選ぶべき
D. NPVは絶対額で評価する指標である
答え：C
解説：NPVが最大でも、投資規模が大きすぎる場合は他の指標（PI等）も考慮すべきです。資金制約がある場合は特に注意が必要です。
```

### 2.5 逆方向（原因→結果 / 結果→原因）
```
【フェーズ】RETAIN
【タイプ】MULTIPLE_CHOICE または TRUE_FALSE

【テンプレート】
「{結果}となる原因として正しいものはどれか。」
「{原因}の場合、どのような結果になるか。」

【例：NPV】
問題：「割引率を高く設定すると、NPVはどうなるか。」
選択肢：
A. 大きくなる
B. 小さくなる
C. 変わらない
D. 予測できない
答え：B
解説：割引率が高いほど、将来CFの現在価値は小さくなるため、NPVは減少します。
```

### 2.6 ミニケース（短文シナリオ→判断）
```
【フェーズ】EXAM
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「{状況の説明}。この場合、{論点}の観点から最も適切な判断はどれか。」

【例：NPV】
問題：「A社は新規設備投資を検討している。初期投資1億円、5年間で毎年3,000万円のCFが見込まれる。割引率10%とした場合、この投資案の判断として正しいものはどれか。」
選択肢：
A. NPVがプラスなので採用すべき
B. NPVがマイナスなので棄却すべき
C. 情報が不足しており判断できない
D. 初期投資が大きすぎるので棄却すべき
答え：A
解説：NPV = 3,000万×3.791（年金現価係数）- 10,000万 ≒ 1,373万円 > 0 なので採用すべきです。
```

## 3. 二次試験向けテンプレート

### 3.1 与件根拠（根拠箇所の選択/指摘）
```
【フェーズ】EXAM
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「与件文に基づき、{論点}の根拠となる記述として最も適切なものはどれか。」
```

### 3.2 設問分解（問われている要素の列挙）
```
【フェーズ】EXAM
【タイプ】SHORT_TEXT

【テンプレート】
「設問『{設問文}』で問われている要素を列挙せよ。」
```

### 3.3 骨子選択（最適な骨子を選ぶ）
```
【フェーズ】EXAM
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「{設問}に対する解答骨子として最も適切なものはどれか。」
```

### 3.4 NG指摘（ダメな理由を選ぶ）
```
【フェーズ】EXAM
【タイプ】MULTIPLE_CHOICE

【テンプレート】
「以下の解答案の問題点として最も適切なものはどれか。」
```

## 4. 問題生成時の注意事項

### 4.1 著作権
- 過去問の問題文をそのまま転載しない
- 出典を明記する場合は引用ルールを守る
- オリジナルの問題文を作成する

### 4.2 解説の書き方
- 「なぜ間違えやすいか」を含める
- 正解の根拠を明確に示す
- 関連する他の論点への言及も検討

### 4.3 難易度のバランス
- 各論点につき、フェーズごとに1問以上
- 難しい問題だけ、簡単な問題だけにならないよう注意
