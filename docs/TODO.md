## セットアップ手順
1. `.env`にNeonのDATABASE_URLを設定
2. `npm install`
3. `npx prisma db push` でスキーマ反映
4. `npm run db:seed` でサンプルデータ投入
5. `npm run dev` で開発サーバー起動（ポート40000）
