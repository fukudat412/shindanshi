# コンテンツポリシー

このドキュメントは、学習コンテンツ（記事・クイズ）作成時の著作権・引用ルールを定めます。

---

## 1. 基本方針

- **オリジナルコンテンツを優先**する
- 過去問の問題文をそのまま転載しない
- 引用する場合は適切な出典を明記する
- 著作権侵害のリスクがある場合は使用しない

---

## 2. 許可されるコンテンツ

### 2.1 オリジナル問題
- 自作の問題文・選択肢・解説
- 公知の事実に基づく問題（法令、会計基準、経営理論など）
- テンプレートに沿って生成した問題

### 2.2 引用が許可されるもの
- 法令・省令・告示（著作権なし）
- 会計基準・監査基準（出典明記）
- 公的機関の統計データ（出典明記）
- 学術論文の一部（引用要件を満たす場合）

### 2.3 参考にしてよいもの
- 過去問の「論点」や「テーマ」
- 試験で問われる「考え方」や「判断基準」
- 一般的な経営理論・フレームワーク

---

## 3. 禁止されるコンテンツ

### 3.1 著作権侵害
- 過去問の問題文をそのまま転載
- 市販テキストの文章をコピー
- 他サイトの問題・解説の無断転載

### 3.2 その他禁止事項
- 特定の個人・企業を誹謗中傷する内容
- 差別的な表現
- 虚偽の情報

---

## 4. 引用のルール

引用する場合は以下を守ること：

1. **必然性**: 引用が必要な理由があること
2. **主従関係**: 自分の文章が「主」、引用が「従」
3. **明確な区別**: 引用部分を「」や引用ブロックで明示
4. **出典明記**: 著者名、書籍名/URL、発行年を記載
5. **改変禁止**: 原文を勝手に変えない

### 出典の書き方例

```
出典：中小企業診断士試験 令和5年度 第1次試験 企業経営理論 第10問（一部改変）
```

```
参考：ポーター, M.E.（1980）『競争の戦略』ダイヤモンド社
```

---

## 5. 問題作成時のチェックリスト

問題を作成・登録する前に確認：

- [ ] 問題文は自作またはテンプレートから生成したものか
- [ ] 過去問の文章をそのままコピーしていないか
- [ ] 引用がある場合、出典を明記したか
- [ ] 選択肢・解説もオリジナルか
- [ ] 特定の個人・企業を傷つける内容がないか

---

## 6. 違反した場合

- 著作権侵害の恐れがあるコンテンツは即時削除
- 繰り返し違反する場合は投稿権限を停止
